# 记忆文档

> **本仓库所有代码均由 AI 生成，作者无编程经验，通过 AI 辅助学习 Python。**

---

## 学习背景

- 作者在正式学习前无任何编程基础
- 学习方式：向 AI 提需求 → AI 生成代码 → 逐行学习理解 → 自行修改测试
- 作者本职非软件工程师，学习 Python 的目的不是成为开发者
- **核心目的：能读 GitHub 上别人的项目代码，看懂别人的代码和思路。** 这是深度使用 AI（Vibe Coding、AI Agent）的基础——读不懂代码，就无法真正判断 AI 输出的质量，也无法理解 AI 工具的原理。
- **要深度不要浅度。** 作者明确要求不要简化内容，不要因为"零基础"就停留在表面。目标是扎实理解，直到能独立浏览和阅读开源项目。
- Python 是理解 AI 工作原理的"窗口"，而非职业方向

## 学习历程

### 2026-07-02：第一个脚本

**知识点：** `open()` / `with` / `for` 循环 / `if` 判断 / `def` 函数 / `import` / 字典列表 / API 调用 / `glob` 文件搜索 / 函数拆分思想

**成果：** `reader.py` — 批量读取 raw_data 中的 .txt 文件，调用 AI 提取卖点，逐个保存到 results/ 并生成 CSV 汇总

---

### 2026-07-03：函数拆解 + 错误处理

**知识点：** `try/except` / `input()` / `os.listdir()` / `datetime` / `timedelta` / `sorted()` / `lambda` / `set()` / `sys.argv` / JSON 读写 / `if __name__ == "__main__"`

**成果：** `daily_work_log.py` — 工作日志系统，支持手动记录、周报生成、自动扫描 Obsidian 日记

---

### 2026-07-06：基础查漏补缺

**知识点（按学习顺序）：**

1. **变量类型** — `int` / `str` / `bool` / `None` / `type()` / 类型转换
2. **运算符与条件判断** — `==` vs `is` / `and or not` / 真值表 / `.get()` / 解包 / `f-string !r`
3. **字符串操作** — `.split()` / `.join()` / `.find()` / `.replace()` / `.strip()` / 切片
4. **列表推导式与 enumerate** — 推导式结构 / `enumerate(start=)` / 组合使用

**成果：** 配套教学脚本 `learn_type_basics.py` / `learn_operators.py` / `learn_string_tools.py` / `learn_list_enum.py`

---

### 2026-07-06：函数进阶与模块（Day14）

**知识点：** 位置参数 vs 关键字参数 / 参数默认值 / `*args` 可变位置参数 / `**kwargs` 可变关键字参数 / 强制位置参数 `/` / 命名关键字参数 `*` / 模块组织与管理 / 内置函数汇总

**成果：** `learn_func_module.py` — 函数参数进阶与模块组织演示

---

### 2026-07-08：函数高级应用（Day17）

**知识点：**
- 装饰器：`@` 语法糖原理 / `wrapper` 与原函数参数保持一致 / 通用版 `*args, **kwargs` / 计时、权限、打折实战案例 / 多层装饰器叠加
- 递归：终止条件 + 递推公式 / 阶乘 / 斐波那契 / 遍历文件夹 / 记忆化缓存优化
- 关键认知：`def` 是注册不是执行 / 代码行号顺序 ≠ 执行顺序 / 好例子降低理解成本

**成果：** `learn_day17.py` — 装饰器 + 递归 教学脚本

---

### 2026-07-09：装饰器深入 + Python 语言特性

**知识点：**
- `__name__` 内置属性（双下划线 = 系统保留）
- 点号 `.` = "从对象里取"
- `time.time()` 前后夹击计时模式
- 函数是一等公民（可赋值、可传参、可返回）
- 装饰器外壳 `timer(func)` 的必要性（工厂 + 闭包）
- `return func(*args, **kwargs)` 透传返回值
- `getattr(obj, "attr", default)` 安全取值
- 函数对象动态绑属性
- 多层装饰器：从下往上套、从上往下执行
- `print(x, end="")` 不换行输出

**成果：** `学习笔记_2026-07-09.md`
