# Python 学习笔记 — 变量类型

## 四种基本类型

| 类型 | 中文 | 装什么 | 能不能算数 |
|---|---|---|---|
| `int` | 整数 | 数字 | ✅ 加减乘除都可以 |
| `str` | 字符串 | 文字（带引号） | ❌ 只能拼接（`+`）和重复（`*`） |
| `bool` | 布尔值 | `True` 或 `False` | 特殊：`True=1`，`False=0` |
| `None` | 空值 | 啥都没有 | ❌ |

## int（整数）

- 数学里的整数，支持所有运算
- `/` 除出来是小数（`10 / 3` → `3.333...`）
- `//` 整除（`10 // 3` → `3`）
- `%` 取余数（`10 % 3` → `1`）
- `**` 乘方（`10 ** 3` → `1000`）

## str（字符串）

- 用引号包起来的都是文字，单引号双引号都行
- `+` 是拼接，不是加法：`"10" + "5"` → `"105"`
- `*` 是重复：`"小明" * 3` → `"小明小明小明"`
- 数字字符串（`"10"`）和整数（`10`）在屏幕上看起来一样，但本质不同

## bool（布尔值）

- 只有两个值：`True` 和 `False`
- 通常来自比较：`10 > 5` → `True`，`10 == 5` → `False`
- **比较符号**：`>`（大于） `<`（小于） `==`（等于） `!=`（不等于） `>=`（大于等于） `<=`（小于等于）
- `=` 是赋值，`==` 才是判断相等
- `if` 后面放的是最终变成 `True / False` 的表达式

## None（空）

```python
user = None       ← 表示"还没有值"
```

- 不是 `0`，不是 `""`，就是纯粹的"空"
- 常用于：查找不到结果、API 请求失败、函数返回"没东西"
- 在 `if` 里相当于 `False`
- `if not user:` = "如果 user 是空的，就执行"

## type() — 查类型

```python
type(10)     → <class 'int'>
type("10")   → <class 'str'>
type(True)   → <class 'bool'>
type(None)   → <class 'NoneType'>
```

**不确定变量是什么类型时，加一行 `print(type(x))`。**

## 类型转换

| 写法 | 作用 |
|---|---|
| `int("123")` | 字符串 → 整数，才能算数 |
| `str(456)` | 整数 → 字符串，才能拼接 |
| `bool(1)` | 非零、非空的值 → `True` |

`"10" + 5` 会报错（字符串和整数不能直接混合操作），必须先转成同一类型。

## if not 的判断逻辑

```python
user = None
if not user:    ← 相当于"如果 user 是空的"
    ...
```

`not` 反转布尔值。None 在 `if` 里相当于 `False`，所以 `not None` = `True`。

## 速查

```
x = 42        → int         ✅ 算数
x = "42"      → str         ❌ 拼接重复
x = True      → bool        特殊
x = None      → NoneType    空
```
