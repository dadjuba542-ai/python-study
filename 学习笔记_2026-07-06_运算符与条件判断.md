# Python 学习笔记 — 运算符与条件判断

## 一、== 和 is 的区别

| 运算符 | 问的问题 | 类比 |
|---|---|---|
| `==` | **内容一样吗？** | 两瓶酒味道一样吗 |
| `is` | **是同一个东西吗？** | 是同一个酒瓶吗 |

```python
a = [1, 2, 3]
b = [1, 2, 3]
a == b  → True   # 两瓶酒味道一样
a is b  → False  # 但是两个不同的瓶子

c = a            # c 指向 a 那个瓶子
a is c  → True   # c 和 a 是同一个瓶子
c.append(4)      # 改 c → a 也跟着变
```

**核心教训：`is` 只用来看 `None`，其他全用 `==`**

```python
result is None     ← ✅ 推荐
result == None     ← ❌ 也行但不推荐
x == 256           ← ✅ 比较数值用 ==
x is 256           ← ❌ 别用
```

**不需要知道小整数缓存**——那不是你需要记的东西，记住"判断值用 `==`，判断空用 `is None`"就够了。

## 二、and / or / not

### and —— 两个都要 True

```python
age > 18 and has_ticket     # 成年 并且 有票 → 才能进
```

### or —— 一个 True 就行

```python
is_vip or has_ticket         # 会员 或者 有票 → 都能进
```

### not —— 反过来

`not` 必须放前面，和英语语序不同：

```python
not is_vip                   # ✅ Python 写法
is_vip not                   # ❌ 报错
not True → False
not False → True
```

### 组合使用

```python
(age > 18 or is_vip) and has_ticket
# (True or False) and True → True and True → True
```

可以用括号控制优先级，和数学一样。

## 三、真值表

以下 **7 种值**在 `if` 里相当于 `False`：

| 值 | 类型 | `bool(值)` |
|---|---|---|
| `None` | 空 | `False` |
| `0` | 整数零 | `False` |
| `0.0` | 小数零 | `False` |
| `""` | 空字符串 | `False` |
| `[]` | 空列表 | `False` |
| `{}` | 空字典 | `False` |
| `()` | 空元组 | `False` |

其余**所有值**都相当于 `True`：

```python
bool(1)      → True
bool(-1)     → True
bool("abc")  → True
bool([0])    → True     # 注意：有内容的列表就是 True
```

## 四、字典的 .get() —— 安全取值

```python
user = {"name": "小明"}
# 没有 is_admin 这个键

user["is_admin"]          # ❌ KeyError 报错
user.get("is_admin")      # → None（不报错）
user.get("is_admin", False)  # → False（找不到就返回默认值）
```

`.get()` 是"防翻车"用的——不确定键一定存在时用它代替 `[]`。

## 五、多级判断的设计

```python
# 从上往下检查，碰到第一个满足的就返回
if 条件 A:
    return "结果 A"
if 条件 B:
    return "结果 B"
if 条件 C:
    return "结果 C"
return "默认结果"    # 全都不满足才到这里
```

**顺序决定优先级。** 条件 A 排第一就意味着它优先级最高。

## 七、f-string 的 !r

```python
v = ""
print(f"{v}")    # →    （看不见，因为空）
print(f"{v!r}")  # → '' （看到一对引号，知道是空字符串）
```

`!r` = "用原始格式显示"。教学时用来让你清楚看到值的真实类型，避免混淆空字符串和 None。

## 八、速查

```
==          → 内容一样吗？（全部都用这个）
is          → 是同一个东西吗？（只用来判断 None）
is None     → 判断是否为空的推荐写法

and         → 两个都要 True
or          → 一个 True 就行
not         → 反过来（放前面）

.get(键, 默认值)  → 安全取值，找不到不报错
```
