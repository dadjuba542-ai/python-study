## 核心内容

### 函数的基本结构

```python
def 函数名(参数):
    """说明文档"""
    # 处理逻辑
    return 结果
```

- `def` = define，定义函数
- `return` = 把结果送回调用处，同时函数结束
- 没有 `return` 的函数返回 `None`

### 参数的四种类型

| 类型 | 写法 | 说明 |
|------|------|------|
| 位置参数 | `def f(a, b)` | 按顺序传值 |
| 关键字参数 | `f(a=1, b=2)` | 指名道姓传值，顺序随意 |
| 默认值参数 | `def f(a=1)` | 不传就用默认值（必须放最后） |
| 可变位置参数 | `def f(*args)` | 收任意多个位置参数 → 元组 |
| 可变关键字参数 | `def f(**kwargs)` | 收任意多个关键字参数 → 字典 |

### 强制位置参数 / 和命名关键字参数 *

- `/` 前面：只能按位置传，不能写参数名
- `*` 后面：只能写参数名传，不能按位置
- 例子：`def f(a, b, /, c, *, d)` — a,b 只能位置；c 两者都行；d 只能关键字

### 函数设计思路

1. **函数就是流水线上的一个工位**：输入 → 处理 → 输出
2. **职责单一**：每个函数只做一件事
3. **先写文档再写代码**：想清楚输出什么再动手
4. **消除重复代码**：一次封装，到处复用

### 模块管理

- 一个 `.py` 文件就是一个模块
- 同名函数放在不同模块里，用 `模块名.函数名` 区分
- 导入方式：
  - `import 模块名` → `模块名.函数名()`
  - `from 模块名 import 函数名` → 直接使用
  - `import 模块名 as 别名` → `别名.函数名()`

## 今日新接触的概念

### `range` vs `randrange`

- `range(n)` — 确定性，生成 `[0, 1, ..., n-1]`
- `randrange(a, b)` — 随机性，从 `[a, b)` 中随机挑一个

### 类型检查

`if type(val) in (int, float):` — 检查参数类型，防止错误数据进入计算

## 数学拾遗

**组合数 C(m, n)**：从 m 个不同东西里选 n 个，有多少种选法。

公式：C(m, n) = m! / (n! × (m-n)!)

例：C(7, 3) = 7! / (3! × 4!) = 5040 / (6 × 24) = 35

## 速查运算符缩写

| 缩写 | 等价于 |
|------|--------|
| `x += y` | `x = x + y` |
| `x -= y` | `x = x - y` |
| `x *= y` | `x = x * y` |
| `x //= y` | `x = x // y` |
