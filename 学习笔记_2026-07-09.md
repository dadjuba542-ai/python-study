## 1. `__name__` 属性

- 每个函数自动带一个 `__name__` 属性，存的是函数名字符串
- 双下划线包起来 = "Python 系统自带的，不是你起的名字"

## 2. 点号 `.` 的作用

- 读作"取"：`对象.属性` → 从对象里取属性/方法

## 3. 装饰器前后夹击模式

```python
def wrapper(*args, **kwargs):
    start = time.time()              # 调用前做的事（记开始时间）
    result = func(*args, **kwargs)   # 实际调用原函数
    end = time.time()                # 调用后做的事（记结束时间+打印）
    print(f"耗时：{end - start}")
    return result                    # 透传返回值
```

- `time.time()` 返回当前时间的秒数（精确到小数点后 6 位）
- `end - start` = 运行耗时

## 4. 函数是一等公民

函数能做的和数字/字符串一样多：
- 赋值给变量
- 放进列表
- 传给另一个函数作为参数

## 5. 装饰器外壳 `timer(func)` 的意义

- `@timer` → `slow_add = timer(slow_add)`
- `timer` 是一个**工厂**，用来"抓住"原函数
- `timer` 必须返回一个**函数**（`wrapper`），而不是执行结果，这样才能配合 `@` 语法
- 如果没有 `timer` 外壳，`wrapper` 里就不知道 `func` 是什么

## 6. `return func(*args, **kwargs)` 的作用

- 不加 `return`：原函数照样执行，但返回值丢在半路，调用方拿到 `None`
- 加 `return`：原函数返回什么，就透传什么

## 7. `getattr(obj, "attr", default)`

- 安全取值函数，替代 `obj.attr`
- 对象上没有该属性时返回默认值，不报错
- 属性名必须传**字符串**类型

## 8. 函数对象绑属性

- Python 里函数也是对象，可以直接往上绑自定义属性
- `wrapper.logged_in = True` → 给函数贴个"已登录"标签

## 9. 多层装饰器叠加

```python
@bold          # 外层
@italic        # 内层（先被装饰）
def get_text():
    return "你好"
```

- 等价于：`get_text = bold(italic(get_text))`
- 执行顺序：**从下往上套，从上往下执行**

## 10. `print(x, end="")`

- `print()` 默认结尾加换行符 `\n`
- `end=""` 把结尾换成空字符串 → 不换行
- 配合 `print()` 空调用补一个换行回来
