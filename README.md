# Python Study

Python 零基础学习仓库。所有代码由 AI 生成，作者通过 AI 辅助逐步学习编程。

> 学习历程、知识点清单详见 [`MEMORY.md`](MEMORY.md)

---

## 应用脚本

| 脚本 | 功能 |
|------|------|
| `reader.py` | 读取 `raw_data/` 中的 .txt → 调 AI 提取卖点 → 逐个保存 + CSV 汇总 |
| `daily_work_log.py` | 工作日志系统：手动记录 / 周报生成 / 自动扫描 Obsidian 日记 |

## 教学脚本

| 脚本 | 对应知识点 |
|------|-----------|
| `learn_type_basics.py` | 变量类型（int/str/bool/None） |
| `learn_operators.py` | 运算符与条件判断（==/is/and/or/not） |
| `learn_string_tools.py` | 字符串操作（split/join/find/replace/切片） |
| `learn_list_enum.py` | 列表推导式与 enumerate |
